UofT Adventure Game - Help

Game Commands

  • Go [Direction] - Move to a different location (north, south, east, west)
  • Take - Take an item from the current location
  • Deposit - Deposit/use an item from your inventory at the current location
  • Drop - Drop an item from your inventory at the current location
  • Undo - Reverse the most recent command and any associated changes to the game
  • Log - View list of all the events (locations visited, commands chosen) in order
  • Help - Display this help page
  • Quit - Quit the game and return to the start screen

Game Rules

  • Your stamina starts at 100 and decreases when you move or interact with items
  • If your stamina reaches 0, the game ends
  • You need to achieve a score of 100 to win
  • Your inventory can initially hold 5 items
  • Some items have special effects when used in the correct location
  • To win, you need to fix your laptop charger, achieve a score of 100, and return to your dorm

Stamina Usage

  • Moving between locations: -3 stamina
  • Taking an item: -4 stamina
  • Using an item: -2 stamina (may vary for special items)

Special Items

  • Free Coffee Coupon: Restores stamina to full when used at Sidney Smith Cafe
  • Lost Backpack: Increases inventory size from 5 to 7 when returned to UofT Bookstore
